# Death Counter
## A simple Spigot plugin that tracks deaths.

# Installation
- Download .jar from releases.
- Place the jar in /plugins
- Start the server.

# Commands
## /deaths (Player)
- Show the number of deaths.

## /set_deaths (Player) Amount
- Set the amount of deaths for a player.

## /death_id (Player) id
- Show data of a death.


# Examples
## /deaths ctih
## /set_deaths ctih 4
## /death_id ctih 2
